üìö Regex Pattern Tips

  • ^ - Start of string anchor
  • $ - End of string anchor
  • \d - Any digit (0-9)
  • + - One or more occurrences
  • ? - Zero or one occurrence
  • (?:[A-Z\d]+_)? - Optional capital letters and/or digit(s) followed by underscore
  • | - OR operator (match either pattern)

Common patterns: ^[A-Z]{3}_TWR$ (any 3-letter airport tower), ^OAK_.*$ (any OAK position)

About Notifications

  • Main Facility Online: When your configured main facility controllers come online
  • Supporting Controllers: When your configured supporting controllers come online
  • All Offline: When all your monitored controllers go offline
  • Errors: When there are issues with the monitoring system

Default Configuration
